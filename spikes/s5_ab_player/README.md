# S5 — toggle A/B player prototype

Build and run from the repo root: `swiftc -o spikes/s5_ab_player/abplayer spikes/s5_ab_player/ABPlayer.swift -framework AVFoundation -framework AppKit && ./spikes/s5_ab_player/abplayer`. A window opens playing A = `source60_preview.mp4` (raw scan) with B = `../s3_pipeline/out60_preview.mp4` (restored) running muted and host-time-anchored underneath. What to check: press SPACEBAR during playback to flip A/B — the swap must be instant with no black flash and no position jump (it only toggles layer opacity, per ADR-8); press P to pause (snaps both players to a frame boundary), then LEFT/RIGHT to frame-step — A and B must always show the same frame (zero-tolerance seeks with the 1001/24000 frame duration); P again resumes both in lockstep. The title bar reads "A (source)" or "B (restored)" so you always know which layer is visible. Pass = no visible glitch on flip, frame-accurate stepping.
